# 基础
## position:relative定位
``` html
<div>
	<div style="position:absolute">
	</div>
  </div>
```
> 第二个div绝对定位是相对于它的父级元素

## 图片截取
``` css
/*-30px用来调整左右间距，-20px用来调整上下间距*/
background:url(imgUrl) no-repeat -30px -20px;
```
