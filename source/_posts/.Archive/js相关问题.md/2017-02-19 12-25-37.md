# 闭包考察
``` js
//输出结果？？如何让其恢复正常
for (var i=0; i<10; ++i) {
    setTimeout( function timer(){
    console.log( i );
    },1000 );
}
//
```
> 解决方案:
``` js
//第一种
for (var i=1; i<=9; i++) {
 (function(j){
     setTimeout( function timer(){
         console.log( j );
     }, 1000 );
 })( i );
}
```
