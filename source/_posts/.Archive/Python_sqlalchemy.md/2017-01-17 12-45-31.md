# 连接数据库
首先需要导入 ` sqlalchemy `库，然后建立数据库连接，这里使用 ` mysql` 通过` create_engine `方法进行.
``` python
from sqlalchemy import create_engine
engine = create_engine("mysql://root:@localhost:3306/webpy?charset=utf8",encoding="utf-8", echo=True)
```
