# 定义model
``` python
class User(Base):
    #表名
    __tablename__='user'
    #表结构
    id = Column(Integer,primary_key=True)
    name = Column(String(20))
    username = Column(String(20))
    password = Column(String(20))
```
> 用法如下
``` python
user=User(name='hhh',username='cs781176645',password='123456')
```
> 不用再多此一举，创建一个User对象
